# 每日清单

## 每天必做（5分钟）

1. **提交 GSC**：打开 Google Search Console → URL 检查 → 提交当天新页面 URL（配额约10个/天）
2. **发 Pinterest pin**：3-5 个 pin，用 docs 里的 image prompts
3. **检查 Vercel Analytics**：看有没有 Pinterest 引荐流量

## 每天内容（问 AI 做）

跟 AI 说 **"今天做第 X 篇"**，AI 会生成页面文件，你审核后推到 Vercel。
